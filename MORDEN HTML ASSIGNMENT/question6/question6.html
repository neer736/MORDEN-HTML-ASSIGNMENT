HTML VS HTML5

HTML HTML5
Previous versions of HTML required longer and more complex doctype declarations. HTML5 uses the simplified doctype declaration (!DOCTYPE html) at the beginning of the document.
it required plugins(third party application)to support audio and video. HTML5 provides built-in support for multimedia such as audio and video tag without any plugins.
it provides less semantic elements so could not provide web accessibility features. it supports more semantic elements such as ("header") ("footer") ("nav")("article") and many more for improving accessibility.
Does not allow JavaScript to run in browser.. Allows JavaScript to run in background. This is possible due to JS Web worker API in HTML5..
it has fewer input elememts and attributes of form compareds to HTML5 it introduced new elements of input in form like : time, date, color, autofocus and attributes like patteren , required etc.
It is almost impossible to get true GeoLocation of user with the help of browser. In Html5 track the GeoLocation of a user easily by using JS GeoLocation API.
